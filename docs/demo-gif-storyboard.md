# Demo GIF Storyboard — WhatsApp to Multi-Agent Firm

## Goal

Create a 20-35 second GIF showing viral value in one glance:

- user sends one business brief
- CEO orchestrates automatically
- 3 departments answer in parallel

## Storyboard (shot-by-shot)

1. Chat input (WhatsApp/Telegram/Discord style)
   - Message: "CEO: Prépare une campagne marketing B2B IA pour Q2, budget 50k, délai 2 semaines."

2. CEO orchestration panel
   - Show routing decision and selected departments
   - Marketing, Strategy, Operations

3. Parallel responses
   - 3 answer cards appear side-by-side or in quick sequence
   - each card includes owner, action plan, deliverables

4. Enterprise safety overlays
   - policy check passed
   - audit event written
   - memory context loaded

5. Final executive summary
   - compact output with next actions and timeline

## Capture tips

- Use fixed width terminal/editor area for readable text
- Keep frame transitions < 800ms
- Prefer dark theme + high contrast
- Export at 1280px width, 12-15 fps

## Suggested caption

One message in. Multi-agent enterprise execution out.
