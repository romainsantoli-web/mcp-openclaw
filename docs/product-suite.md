# Product Suite — mcp-openclaw + setup-vs-agent-firm

## Product model

This is one product delivered as two public repositories:

1. mcp-openclaw (generic orchestration core)
2. setup-vs-agent-firm (firm extension with CEO/departments/services)

## Why split this way

- Core can be adopted in any domain
- Firm adapter accelerates enterprise rollout
- Ecosystem contributors can extend without touching core

## Public status (checked)

- https://github.com/romainsantoli-web/mcp-openclaw (Public)
- https://github.com/romainsantoli-web/setup-vs-agent-firm (Public)

## Shared value proposition

Transform OpenClaw into an enterprise multi-agent AI with safe-by-design controls (policy, allowlist, audit, memory, metrics).
